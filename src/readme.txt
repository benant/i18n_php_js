번역 도구는 https://poedit.net/download 에서 다운로드 하거나 https://localise.biz/free/poeditor 에서 사용합니다.

[번역 작업 순서]

1. 현 파일을 실행해 번역이 필요한 문구를 추출 및 정리합니다. window 에서는 git bash 프로그램에서 실행하시면 됩니다.
   www/i18n/build.sh

2. pot 파일을 poedit 프로그램으로 열어서 번역합니다. 그리고 저장할때 번역언어별로 저장합니다. 
   저장경로는 번역 언어명 폴더입니다. 언어_국가코드 형식으로 사용할 수 있습니다.

3. 신규 번역 글 반영
   기존 po 파일에 새로 생성된 pot 파일을 반영하려면 poedit 프로그램에서 po 파일을 열고 상단 매뉴 > 카탈로그 > POT파일에서 업데이트 를 클릭하고 pot 파일을 선택하면 두 파일이 합쳐집니다.

4. mo파일 생성
   실재로 PHP에서 사용하는 번역 파일은 .mo파일입니다. .po 파일을 poedit 프로그램으로 열고 저장하면 .mo 파일도 함깨 생성됩니다. 
   매뉴 > MO로 컴파일 매뉴를 누르면 따로 .mo 파일을 만들 수 있습니다.

5. json파일 생성
	Javascript에서 사용하는 번역데이터파일은 /i18n/library.php 파일에서 접속자의 언어에 맞게 자동으로 생성합니다. 주의할것은 PHP가 Javascript보다 먼저 실행되어야 합니다.

6. PHP 파일에서 번역함수 사용
   PHP에서는 __(), _e() 함수를 사용합니다. __는 출력없이 번역된 값만 받을수 있고 _e는 번역된 값을 바로 화면에 출력(echo)합니다.
   택스트 에디터에서 정규표현식을 이용한 문자열 치환을 사용하면 손쉽게 모든 파일에 번역을 적용할수 있습니다.
   예제)
   검색어: errMsg\((.[^_]*)'(.[^']*)'\)   errMsg\((.[^_]*)"(.[^"]*)"\)  alertGo\('(.[^']*)'  alertGo\("(.[^"]*)"
   치환어: errMsg($1__('$2'))             errMsg($1__("$2"))            alertGo(__('$1')     alertGo(__("$1")

   errMsg(0,'그림인증문자를 입력해주세요.');
   errMsg('그림인증문자를 입력해주세요.');

   아래문자열검색은 조심히 소스를 보면서 치환해야합니다.
   검색어: '(.[가-힇., ]*)'  "(.[가-힇., ]*)"
   치환어: __('$1')       __("$1")

7. HTML 파일에서 번역함수 사용
   HTML 파일에서도 PHP 함수를 사용합니다. 따라서 웹서버는 html 파일도 PHP가 작동하도록 설정해야합니다. tpl 확장자를 사용하면 해당 파일도 php가 작동하도록 설정하면 됩니다.

   검색어: title="(.[^"]*)"			placeholder="(.[^"]*)"
   치환어: title="<?php _e("$1")?>" placeholder="<?php _e("$1")?>"

   value="확인"
   
8. JS 파일에서 번역함수

   검색어: alert\('(.[^']*)'\)	alert\("(.[^"]*)"\)	confirm\('(.[^']*)'\)	confirm\("(.[^"]*)"\)	throw "(.[^"]*)"	'msg':'(.[^']*)'	'name':'(.[^']*)'	html\('(.[^']*)'\)
   치환어: alert(__('$1'))		alert(__("$1"))		confirm(__('$1'))		confirm(__("$1"))		throw __("$1")		'msg':__('$1')		'name':__('$1')		html(__('$1'))

    throw __("CHEditor는 현재 사용 중인 브라우저를 지원하지 않습니다.");
	의 경우 메시지를 표시하기는 하지만 개발관련 내용도 있어서 전부 번역을 적용하지는 않았습니다.

	'msg':__('작성자 이름을 입력하세요.!')
	html(__('아이디를 입력하여 주세요.!'))
